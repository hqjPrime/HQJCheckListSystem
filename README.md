# HQJ 检查列表系统 (HQJ Check-List System)

一个功能丰富的Web待办事项列表管理工具，支持分类管理、智能分类建议、数据统计、导入导出等功能。

## ✨ 功能特性

### 📋 核心功能

- **待办事项管理**：添加、删除、标记完成
- **分类系统**：支持自定义分类和智能分类建议
- **智能学习**：根据关键词自动学习分类规则
- **分类过滤**：按分类筛选待办事项
- **统计面板**：实时显示完成率统计数据
- **时间显示**：精确到毫秒的时间显示
- **导入导出**：支持Markdown格式的数据导入导出

### 🎨 界面特性

- **响应式设计**：适配不同屏幕尺寸
- **流畅动画**：波浪动画、流动条动画、脉冲效果
- **视觉反馈**：完成状态、打钩效果等视觉反馈
- **分类建议提示**：智能提示建议的分类

## 🚀 快速开始

### 环境要求

- 现代浏览器（Chrome、Firefox、Edge、Safari等）
- 无需服务器，纯前端应用

### 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器（如 VS Code Live Server）运行项目

## 📖 使用指南

### 添加待办事项

1. 在输入框中输入待办事项内容
2. 系统会自动建议分类（基于关键词学习）
3. 点击分类建议或使用默认分类
4. 按 Enter 键或点击添加按钮完成添加

### 管理待办事项

- **标记完成**：点击复选框
- **删除项目**：点击删除按钮
- **按分类筛选**：点击顶部的分类筛选按钮

### 智能分类

- 系统会根据关键词自动学习和建议分类
- 首次使用时，系统会从 `keywords.json` 加载预定义的关键词映射
- 新添加的项目会自动关联对应的分类

### 数据管理

- **导出数据**：点击导出按钮，生成Markdown格式的文件
- **导入数据**：点击导入按钮，选择Markdown文件导入
- **清空数据**：点击清空按钮，清除所有待办事项

## 🔧 技术实现

### 核心模块

- **状态管理**：使用全局变量管理待办事项、分类、筛选状态
- **本地存储**：数据持久化存储（使用 localStorage）
- **关键词学习**：动态学习和更新关键词-分类映射
- **渲染引擎**：高效的列表和统计面板渲染
- **动画系统**：CSS动画实现波浪、流动、脉冲等效果

### 依赖说明

- 无外部依赖库
- 纯原生 JavaScript + CSS 实现
- 使用 HTML5 本地存储进行数据持久化

## 🎯 核心函数说明

| 函数名                   | 功能描述                  |
| ------------------------ | ------------------------- |
| `init()`                 | 初始化应用，加载数据      |
| `addItem()`              | 添加新的待办事项          |
| `toggleDone()`           | 切换待办事项的完成状态    |
| `deleteItem()`           | 删除待办事项              |
| `filterByCategory()`     | 按分类筛选待办事项        |
| `learnCategory()`        | 学习并记录关键词-分类映射 |
| `getSuggestedCategory()` | 获取智能建议的分类        |
| `exportToMarkdown()`     | 导出数据为Markdown格式    |
| `importFromMarkdown()`   | 从Markdown文件导入数据    |
